Javascript Messenger Personal Project

This is a Javascript messenger project started on January 11th, 2020. Its final version was completed on January 17th, 2020. This is a front-end inner HTML messenger application. The application uses HTML, CSS, and Javascript. The three buttons below allow you to add or remove messages posted to the screen from what is typed in the text box. The boneworks button gives you an alert message with a quote from the Virtual Reality game Boneworks. More detailed documentation can be viewed from my GitHub respository. The application and website were coded in Visual Studio Code.
